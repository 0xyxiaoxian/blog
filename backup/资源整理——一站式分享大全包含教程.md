关于激活命令的补充，安装后显示未激活，使用以下命令来激活 
```
cd C:\Program Files\Microsoft Office\Office16 
slmgr /skms kms.03k.orgs 
lmgr/ato 
```
**注意：以上命令需要以管理员身份运行cmd来执行。执行后弹出提示已经激活完成就可以了。此时需要重启系统，就显示激活了。**
